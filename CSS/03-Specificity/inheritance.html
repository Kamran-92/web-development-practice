inheritance

In CSS, inheritance is the behavior where certain CSS properties applied to a parent element are automatically passed down

1. Properties that DO inherit naturally

These are usually properties related to typography and text. It makes sense because if you set the font for an article, you usually want all the paragraphs and lists inside it to use that same font.

  1. color
  2. font-family
  3. font-size
  4. font-weight
  5. line-height
  6. text-align

2. Properties that DO NOT inherit naturally

These are usually properties related to the box model and layout. If these inherited automatically, it would cause chaos. Imagine if you put a border around a main container, and suddenly every single paragraph, link, and image inside it also had that exact same border!

  1. margin
  2. padding
  3. border
  4. background-color / background-image
  5. width / height
  6. display
