Specificity-Order

CSS specificity is the algorithm browsers use to decide which style rule applies when multiple CSS rules target the exact same HTML element. It calculates a mathematical "weight" based on the selector types used, prioritizing inline styles highest, followed by IDs, then classes/pseudo-classes, and finally base elements.
